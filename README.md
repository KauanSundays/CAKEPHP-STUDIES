# CakePHP Basics Repository
Este repositório é uma coleção de exercícios e projetos concebidos para familiarizar os desenvolvedores com os fundamentos do framework CakePHP. Constitui uma base sólida para quem busca aprofundar seus conhecimentos em desenvolvimento web com CakePHP.

### Exercícios
<h2>Rotas e Controladores
  <ol>
    <li>Crie uma rota que retorne uma mensagem simples. ✔️</li>
    <li>Defina uma rota que aceite um parâmetro de URL e o exiba. ✔️</li>
    <li>Crie o controlador chamado UsersController. ✔️ </li>
    <li>Defina rotas para listar e exibir usuários utilizando o controlador. ✔️ </li>
    <li>Adicione uma rota que redireciona para outra rota. ✔️ </li>
    <li>Utilize a diretiva foreach do CakePHP para exibir uma lista de itens em uma view. ✔️ </li>
  </ol>
<h2>Modelos e Banco de Dados
  <ol>
     <li>Crie um modelo chamado Product utilizando o bake. ✔️ </li>
     <li>Migre uma tabela de banco de dados para armazenar produtos. ✔️ </li>
     <li>Crie um seeder para popular a tabela de produtos com dados de exemplo. ✔️</li>
     <li>Escreva uma consulta ORM para recuperar todos os produtos. ✔️ </li>
     <li>Adicione validação de dados ao criar um novo produto. ✔️ </li>
     <li>Exiba a lista de produtos em uma view. ✔️ </li>
  </ol>
<h2>Autenticação
  <ol>
    <li>Configure a autenticação de usuário usando bake.  </li>
    <li>Crie um sistema de registro de usuário.  </li>
    <li>Implemente o login de usuário.  </li>
    <li>Proteja uma rota para que apenas usuários autenticados possam acessá-la.  </li>
  </ol>
<h2>Formulários e Validação
  <ol>
    <li>Crie um formulário para criar um novo produto. ✔️ </li>
    <li>Valide os dados do formulário no controlador.  </li>
    <li>Exiba mensagens de erro na view quando a validação falhar.  </li>
  </ol>
<h2>Views e Templates
  <ol>
    <li>Crie uma view que estenda um layout principal.</li>
    <li>Utilize diretivas if e else do CakePHP para controle condicional.</li>
    <li>Utilize a diretiva foreach para exibir uma lista de produtos.</li>
    <li>Crie um arquivo de layout personalizado para suas views.</li>
  </ol>
<h2>Relacionamentos em ORM
  <ol>
    <li>Defina um relacionamento de um-para-muitos entre um modelo Post e Comment.</li>
    <li>Crie um formulário para adicionar comentários a um post.</li>
    <li>Exiba os comentários associados a um post em sua view.</li>
  </ol>
<h2>Paginação
  <ol>
    <li>Divida uma lista de produtos em várias páginas usando paginação.</li>
    <li>Adicione links de paginação à sua view.</li>
  </ol>
<h2>Envio de E-mails
  <ol>
    <li>Configure o envio de e-mails usando o serviço de envio de e-mails do CakePHP.</li>
    <li>Crie um controlador para enviar um e-mail de contato.</li>
    <li>Use uma view para criar o conteúdo do e-mail.</li>
  </ol>
<h2>Fila (Queue)
  <ol>
    <li>Configure a fila para processar o envio de e-mails em segundo plano.</li>
    <li>Crie uma tarefa de fila para enviar e-mails.</li>
    <li>Dispare a tarefa de fila quando um usuário enviar um formulário de contato.</li>
  </ol>
<h2>Agendamento (Scheduling)
  <ol>
    <li>Configure uma tarefa agendada que é executada a cada dia.</li>
    <li>Use o agendamento para realizar tarefas como limpar registros antigos.</li>
  </ol>
<h2>Gerenciamento de Erros
  <ol>
    <li>Personalize a página de erro 404.</li>
    <li>Implemente uma página de erro personalizada para erros 500.</li>
  </ol>
<h2>Arquivos Estáticos e Asset Mix
  <ol>
    <li>Configure o CakePHP Mix para compilar CSS e JavaScript.</li>
    <li>Adicione arquivos CSS e JS a uma view e compile-os usando Mix.</li>
    <li>Configure Mix para minificar e combinar arquivos.</li>
  </ol>
<h2>Testes
  <ol>
    <li>Escreva um teste de unidade para um método em um controlador.</li>
    <li>Escreva um teste funcional para verificar a autenticação de usuário.</li>
    <li>Execute os testes usando PHPUnit.</li>
  </ol>
<h2>API RESTful
  <ol>
    <li>Crie um controlador e rotas para uma API RESTful simples.</li>
    <li>Use autenticação de token para proteger suas rotas.</li>
    <li>Escreva documentação básica para sua API usando Markdown.</li>
  </ol>
  
## Projetos

Além dos exercícios, este repositório inclui projetos práticos para aplicar os conceitos aprendidos:

<H1> Blog Avançado </H1>

  Desenvolva um sistema de blog mais avançado com recursos como categorias, tags e uma interface de administração.

<H1> Plataforma de E-learning </H1>
  Construa uma plataforma de e-learning com cursos, aulas e perfis de usuários.

<H1> Aplicação de Mídia Social </H1>
  Crie uma aplicação de mídia social com funcionalidades como postagens, comentários e amizades.

<H1> Sistema de Reservas de Eventos </H1>
  Desenvolva um sistema para reserva de eventos, permitindo que usuários agendem participação em eventos.
  
## Como Usar
Clone este repositório e siga as instruções em cada exercício ou projeto. Certifique-se de ter o ambiente de desenvolvimento CakePHP configurado.
